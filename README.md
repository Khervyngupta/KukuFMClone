# KukuFM Landing Page

## Description
This is the improved version of the KukuFM landing page with infinite scroll, auto slider carousel, and a fixed header.

## Features
- Infinite Scroll
- Auto Slider Carousel
- Fixed Header

## Instructions for Running Locally
1. Clone the repository
   ```bash
   git clone https://github.com/your-username/kukufm-landing.git

2. Navigate to the project directory
    cd kukufm-landing

3. Install dependencies
    Install dependencies

4. Run the development server
    npm run dev

5. Open http://localhost:3000 with your browser to see the result.
